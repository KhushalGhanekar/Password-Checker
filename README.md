# 🔐 Password Strength Checker (Java)

A beginner-friendly Java project that evaluates the strength of a password based on important security criteria and provides suggestions for improvement.

---

## 🎯 About

This project was developed as part of my first-year engineering journey to understand basic programming and cybersecurity concepts.

It focuses on analyzing password strength using parameters such as length, character variety, and complexity — which are essential in real-world security systems.

---

## 🚀 Features

- ✔️ Checks minimum password length (8 characters)
- ✔️ Detects uppercase and lowercase letters
- ✔️ Verifies presence of digits (0–9)
- ✔️ Identifies special characters
- ✔️ Classifies password as:
  - Weak 🔴
  - Medium 🟡
  - Strong 🟢
- ✔️ Provides suggestions to improve weak passwords

---

## 🧠 Concepts Used

- Java Basics (Loops, Conditions, Methods)
- String Handling
- Character Validation
- Basic Cybersecurity Principles (Password Policies)
